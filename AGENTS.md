# Project Overview

- `Website 3/` - Old version of the website
- `apex-new-site-main/` - New version of the website

## Goal
Migrate specific elements from the old website (`Website 3`) to the new website (`apex-new-site-main`).

## Rules
1. Only implement what is explicitly requested
2. Minimize bugs, destructive changes, and code conflicts
3. Preserve existing functionality in the new site
